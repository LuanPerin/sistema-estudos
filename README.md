# Sistema de Estudos

Gerenciador de estudos pessoal desenvolvido em Python com Streamlit.

## Funcionalidades
- Cronograma automático
- Registro de estudos com timer
- Controle de revisões
- Relatórios de desempenho
- Login persistente

## Como rodar
```bash
pip install -r requirements.txt
streamlit run Login.py
```
